# Automated Affiliate Deals System

An end-to-end automation system that:
- Collects latest deals
- Prevents duplicates
- Stores data
- Publishes to Telegram
- Uses Python + n8n workflows

### Tech Stack
- Python
- SQLite
- Telegram Bot API
- n8n (workflow orchestration)

### Use Cases
- Affiliate marketing
- Deal aggregation
- Automation engineering demos
